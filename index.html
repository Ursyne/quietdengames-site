The Interval Bureau: Extension 0

Every choice is logged. Every pause is assessed. A 45-minute branching audio narrative where workplace protocol meets cosmic intrusion. Navigate strange menus, enforced breathing exercises, and controlled scenarios that bend cause and effect. The Interval Bureau is listening.

Follow on itch.io Learn more
Audio-first. One desk, one phone, no clutter. Single sitting. Built for ~45 minutes start to finish. Minimal input. Number keys only; the system listens more than you speak.

A contained call that escalates in controlled steps. No filler, no fake choices: the line measures how you follow instructions once they stop feeling safe. Content notes: breath control themes, panic sensations, surveillance, depersonalization; no graphic violence.

About Quiet Den Games

Quiet Den Games is a solo label for sharp, atmospheric interactive work: short runtimes, deliberate writing, and systems that respect your time. Extension 0 is the first release set inside the Interval Bureau.
